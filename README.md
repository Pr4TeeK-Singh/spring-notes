## 📚 Chapter Summaries

### [Chapter 1 — Spring Core](chapters/chapter-1-spring-core.md)
The foundation of the Spring Framework. Covers **IoC (Inversion of Control)**, **Dependency Injection** (constructor and setter), **Beans**, the **Spring Container**, **ApplicationContext**, bean configuration methods (XML, Annotation, Java), important annotations, **bean scopes**, and the **bean lifecycle**.

### [Chapter 2 — Spring MVC](chapters/chapter-2-spring-mvc.md)
Covers building web applications with Spring. Explains the **Model-View-Controller** pattern, the role of **DispatcherServlet** as the front controller, the complete **request-response flow**, and how Controllers handle HTTP requests.

### [Chapter 3 — Spring AOP](chapters/chapter-3-spring-aop.md)
Covers **Aspect Oriented Programming** — the technique of separating cross-cutting concerns (logging, security, transactions) from business logic. Explains key terms: **Aspect**, **Advice**, **Joinpoint**, and **Pointcut**.

### [Chapter 4 — Spring Security](chapters/chapter-4-spring-security.md)
Covers securing web applications with Spring Security. Explains **Authentication** vs **Authorization**, role-based access control, password protection, login/logout security, and security configuration.
